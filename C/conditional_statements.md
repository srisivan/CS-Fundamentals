# Conditional Statements in C

## What
- Control flow: `if`, `else if`, `else`, `switch`

## Syntax
```c
if (x > 0) {
  // do something
} else if (x == 0) {
  // do something else
} else {
  // fallback
}

switch (x) {
  case 1:
    break;
  default:
    break;
}
```

## When to use
- `if/else` for ranges/complex conditions
- `switch` for discrete known values
