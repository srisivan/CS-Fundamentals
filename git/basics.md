# Git Basics

- Distributed version control: tracks snapshots over time.
- Key Commands:
  - `git init`: Initialize repo.
  - `git add`: Stage changes.
  - `git commit`: Save snapshot.
  - `git push` / `git pull`: Sync with remote.
- Branching: Isolate work, merge later.
- `git config`: Set username, email, default branch.
- Check status: `git status`
