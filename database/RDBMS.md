# RDBMS (Relational Database Management System)

- Stores data in related tables (rows and columns).
- Supports ACID properties: Atomicity, Consistency, Isolation, Durability.
- Uses SQL for creating, reading, updating, deleting data.
- Supports constraints: Primary Key, Foreign Key, Unique, Not Null.
- Examples: MySQL, PostgreSQL, Oracle, SQL Server.
