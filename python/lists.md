# Lists in Python

## What is a List?
- Ordered, mutable collection.
- Supports heterogeneous data.

## Common Operations
- Append: `list.append(item)`
- Remove: `list.remove(item)`
- Slicing: `list[1:3]`
- Sort: `list.sort()`

## Example
```python
my_list = [1, 2, 'three']
my_list.append(4)
print(my_list[0:2])
```
