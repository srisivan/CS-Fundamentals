# Dictionaries in Python

## What is a Dictionary?
- Key-value pairs.
- Unordered.
- Keys must be unique and hashable.

## Common Operations
- Access: `dict[key]`
- Get: `dict.get(key)`
- Add: `dict[key] = value`
- Iterate: `for key, val in dict.items()`

## Example
```python
my_dict = {'name': 'Srisivan', 'age': 25}
print(my_dict['name'])
```
